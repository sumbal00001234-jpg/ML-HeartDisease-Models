##  Project Overview
This project compares **5 supervised machine learning models** for heart disease prediction using the Heart Disease dataset from Kaggle.

---

##  Models Implemented

| # | Model | Type | Key Feature |
|---|-------|------|-------------|
| 1 | Logistic Regression | Classification | Binary classification baseline |
| 2 | Support Vector Machine (SVM) | Classification | Handles complex boundaries |
| 3 | Decision Tree | Classification | Easy interpretation |
| 4 | Random Forest | Classification | Ensemble learning |
| 5 | k-Nearest Neighbors (k-NN) | Classification | Instance-based learning |
